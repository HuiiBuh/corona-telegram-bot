# Corona Telegram Bot

Get the current incidence and statistics for Germany.

## Docker

Run the bot on your own server

| Environment Variable |  Default  |  Required  |  
|-----|---|---|
|   BOT_SECRET   |   |  YES  |
|   COVID_API   | https://api.corona-zahlen.org/  |   |
|  DB_LOCATION   | /database  |  NO |
|  PRODUCTION   | True  |  NO  |
|  VERIFY_SSL   | True  |  NO  |
